# Punteros React

Aplicación en React creada con Vite para explicar de forma didáctica los conceptos básicos de los punteros en C++. 
Incluye animaciones con [Framer Motion](https://www.framer.com/motion/) para visualizar cómo un puntero se mueve entre direcciones de memoria.

## Scripts disponibles

- `npm run dev` – inicia el entorno de desarrollo.
- `npm run build` – genera la versión de producción.
- `npm run lint` – ejecuta ESLint.

## Estructura

- `src/components/PointerVisualizer.jsx`: animación interactiva del puntero.
- `src/data/concepts.js`: listado de conceptos sobre punteros.

## Uso

Instalar dependencias e iniciar:

```bash
npm install
npm run dev
```

Luego abrir <http://localhost:5173> en el navegador.
